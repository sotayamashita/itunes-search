# iTunes Search

> Get app information on App Store


## Install

```javascript
$ npm install sotayamashita/itunes-search --save-dev
```


## Usage

```javascript
const itunesSearch = require('itunes-search');

itunesSearch('query').then(data => {
   console.log(data);
});
```
